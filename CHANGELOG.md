# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2025-08-01

### Added
- Integrated advanced LLM search option directly into the global promptbar container.
- Implemented persistent user search history tracking list module.
- Added interactive bookmarking infrastructure and complete custom place registration UI components.
- Created automated connection routes linking the add-new-place action trigger button to backend API endpoints.
- Implemented robust like and dislike transactional logic directly into the reusable UserComment component.
- Added distinct visual loading state indicators across search execution flows.
- Added full localized Persian number representation formatting engine across the website layouts.

### Fixed
- Fixed critical scroll bar clipping and layout containment issues inside image display components.
- Resolved map marker display anomalies where points failed to render when random location nodes were selected.
- Fixed geolocation positioning coordinates math errors triggered when map longitude values entered negative ranges.
- Corrected interface rendering bugs within the star-rating layout module.
- Resolved automated view-flipping sorting directions inside search history panels.

---

## [1.1.0] - 2025-05-20

### Added
- Integrated production continuous integration (CI) automation pipelines utilizing `.gitlab-ci.yml`.
- Created complete authenticated user profile workspace layout alongside dedicated platform sidebar components.
- Established native connection pathways linking user profile views directly with the backend API.
- Integrated parallel routing capabilities supporting advanced multi-view presentation layouts.
- Added interactive public comment streams directly onto destination cards and connected them to core backend services.
- Implemented direct phone calling action routing links targeting system business profiles.
- Added dynamic icon layers leveraging Google Material Symbols across map categories.
- Integrated automated category filtering arrays to suppress unnamed geospatial nodes.

### Fixed
- Overhauled application layout engine to support complete cross-device viewport responsiveness.
- Resolved asynchronous race conditions in the authentication engine by fixing localhost cookie setup procedures.
- Fixed routing validation bugs where explicit coordinate sets ('lat/lng') dropped away from platform URLs during view shifts.
- Resolved layer retrieval exceptions (`getLayer err`) during dense geospatial rendering steps.
- Solved nested dependency injection vulnerabilities by correcting deep packages paths inside lock configurations.
